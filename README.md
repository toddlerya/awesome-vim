# awesome-vim
Pythoner's vim


# Install

#### 1. backup your old .vimrc if it is necessary

```
cp ~/.vimrc ~/.vimrc_bak
```

#### 2. just get the file

recommend
```
git clone https://github.com/toddlerya/awesome-vim.git
ln -s awesome-vim/vimrc ~/.vimrc
```

#### 3. Done, enjoy it
